
# Central Station Management System 

Websocket Server for simple CSMS

## Prerequisite

1. Node Version 12+
2. npm Version 6+

### Connection URL 

`node simulator -s mt11 -u ws://localhost:3002`  
`node simulator -t status_notification -s mt11 -u ws://localhost:3002`

### Run Service 

1. Install dependencies
 
 `npm install`

 or 

 `yarn`

2. Run tests

 `npm test`

 or 

 `yarn test`

3. Run 

 Command line
 `npm start`

 or 
 
 `yarn start`
