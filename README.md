# edrv-ws-solution
# Central Station Management System 

Websocket Server for simple CSMS

## Prerequisite

1. Node Version 12+
2. npm Version 6+

### Run Service 

1. Install dependencies
 
 `npm install`

 or 

 `yarn`

2. Run tests

 `npm test`

 or 

 `yarn test`

3. Run 

 Command line
 `npm start`

 or 
 
 `yarn start`
